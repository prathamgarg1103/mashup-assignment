# Mashup Assignment

Fresh project workspace for the mashup assignment.

## Quick start

1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Rename `ROLLNUMBER.py` to your roll number (example: `101556.py`) before submission.

## Program 1 CLI format

```bash
python <RollNumber>.py <SingerName> <NumberOfVideos> <AudioDuration> <OutputFileName>
```
